Documentacion para Usuarios

Este proyecto consiste en una plataforma web de gestión de incidencias que ofrece una solución integral para administrar y abordar eficazmente problemas y solicitudes. La aplicación se compone de cuatro páginas navegables: una que muestra las incidencias activas, otra para visualizar detalles específicos de cada incidencia, una tercera que permite el registro de nuevas incidencias a través de un formulario intuitivo y, finalmente, una página de documentación que proporciona orientación y recursos para usuarios y administradores. Esta aplicación brinda una interfaz amigable y eficiente para gestionar y resolver incidencias de manera organizada y eficaz.

Página General de Incidencias

La página general de incidencias es la página principal donde los usuarios pueden ver una tabla de todas las incidencias activas. A continuación se describen los elementos clave de esta página:

  1. Tabla de Incidencias Activas:
    • Id de Incidencia
    • Fecha de Apertura
    • Prioridad de la Incidencia
    • Tipo de Incidencia
    • Departamentos Implicados
    • Id del Técnico Asignado
    • Opción para "Resolver Incidencia"
    • Opción para "Acceder a la Incidencia"
  2. Acceder a la Incidencia:
    • Para ver los detalles de una incidencia en particular, los usuarios deben hacer clic en "Acceder a la Incidencia". Esto los llevará a la página específica de esa incidencia.

Página Específica de Incidencias

La página específica de incidencias permite a los usuarios ver los detalles completos de una incidencia en particular. Aquí están los componentes clave de esta página:

  1. Detalles Básicos de la Incidencia:
    • Id de Incidencia
    • Fecha de Apertura
    • Prioridad de la Incidencia
    • Tipo de Incidencia
    • Departamentos Implicados
    • Id del Técnico Asignado
  2. Detalles Avanzados de la Incidencia:
    • En esta zona, los usuarios pueden ver una descripción detallada de la incidencia que proporciona información adicional sobre el problema.

Formulario para Generar una Incidencia

El formulario para generar una incidencia es donde los usuarios pueden agregar nuevas incidencias al sistema. A continuación, se detallan los campos que deben completar:

  1. Fecha de Apertura:
    • Los usuarios deben seleccionar o ingresar la fecha y hora en que se abre la incidencia.
  2. Descripción Corta:
    • Aquí deben proporcionar una breve descripción del problema o incidencia.
  3. Departamentos Implicados:
    • Selecciona los departamentos involucrados en la incidencia.
  4. Tipo de Incidencia:
    • Los usuarios deben indicar el tipo de incidencia. Algunas opciones son "Hardware", "Software", "Problemas de Red" o "Seguridad informatica".
  5. Prioridad de la Incidencia:
    • Los usuarios deben especificar la prioridad de la incidencia. Las opciones son "Crítica", "Alta", "Media", "Baja", "Muy Baja/Nula".
  6. Id del Técnico Asignado:
    • Si se asigna un técnico específico a la incidencia, debe ingresarse su ID.
  7. Credenciales del Usuario:
    • Los usuarios deben proporcionar las credenciales necesarias para verificar su identidad o autorización para informar sobre la incidencia.
